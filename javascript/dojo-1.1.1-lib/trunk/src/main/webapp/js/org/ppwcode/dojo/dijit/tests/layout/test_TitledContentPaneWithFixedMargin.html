TitleContentPane:

Is a pane with a title on top, which can have a shadow.

An accessible container with a Title Heading. TitledContentPane is an
extension to ContentPane. It provides all the ContentPane it's usesful
content-control aspects.


features:

  • Add a title
  • Use a shadow
  • Put a margin on the component

example:

Here you see an example of a TitledContentPane with a title and fixed margin.
If you set the margin, it will set the margin on every side (top, right, bottom and left)
