Address Widget:

Is a pane with a title on top, which can have a shadow.

An accessible container with a Title Heading. TitledContentPane is an
extension to ContentPane. It provides all the ContentPane it's usesful
content-control aspects.


features:

  • Country is filled with live data from the DB through a dataStore
  • Depending of the country there will be a check on the regex check on the postal code

Here you see an example of the Address Widget.
