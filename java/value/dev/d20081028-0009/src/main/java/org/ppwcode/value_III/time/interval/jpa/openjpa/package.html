OpenJPA value handlers for effective time intervals.

Sadly, JPA as a standard does not support "user types" or "custom types" or "value types" as any reasonable other technology does, from Hibernate over OpenJPA, but also including Struts, JSF, dwr, json-lib, and many, many others. Therefor, we introduce the OpenJPA solution, which can be used in OpenJPA contexts, and other solutions for other contexts.
